# laravel_streaming_film
laravel 9 app film streaming

# Technology used
- Laravel 9
- Livewire
- TailwindCSS
- MySQL
- ReactJS

# Requirements
- PHP >= 8.1
- Composer
- NodeJS
- NPM
- MySQL
-

# Installation
- Clone this repo
- Run `composer install` or `composer update`
- Run `npm install`
- Run `npm run dev`
- Run `cp .env.example .env`
